# Pixie Customer Guidance Repository

Welcome to the official **Pixie Customer Guidance** repository. This guide is designed to help you seamlessly set up and operate Pixie, your autonomous delivery robot. The documentation is organized into several sections, each covering a distinct aspect of handling and utilizing Pixie effectively:

1. **Pixie Unboxing**: This section describes how to properly unbox Pixie and set it up for the first time, ensuring that all components are handled safely and correctly prepared for initial use.

2. **ROS Manual**: Provides comprehensive instructions on how to record and clean the map, define routes, and address operational issues. This manual is essential for configuring Pixie to navigate your site effectively.

3. **Access Commands**: Introduces how to operate Pixie using three methods, ranging from basic manual controls to advanced cloud-based commands. This section ensures that users of all technical levels can interact with Pixie efficiently.

4. **Cloud-Based Manuals**: Describes an example of creating a cloud instance and integrating it with Pixie. This section will guide you through setting up a cloud environment that can communicate with and control Pixie remotely, enhancing its capabilities and your convenience.

Please note that these folders are organized here based on their natural implementation order to facilitate a smooth setup and operational experience.
