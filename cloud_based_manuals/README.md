# Cloud-Based Robot Dashboard Implementation

This repository contains all the necessary files to set up and implement a cloud-based robot dashboard using AWS and Node-RED.

## Included Files

1. **Node-RED Flow File**
   - File: `mvp_dashboard_flows.json`
   - Description: JSON file containing the Node-RED flow configuration for the dashboard.

2. **Node-RED AWS Instance Documentation**
   - File: `nodered_AWS_instance.md`
   - Description: Instructions for setting up a Node-RED instance on AWS.

3. **Robot AWS Instance Documentation**
   - File: `robot_AWS_instance.md`
   - Description: Guide for creating and configuring an AWS instance for the robot.

Please refer to these files for detailed setup and implementation instructions.
